# Kaiden Bell — Portfolio

A modern personal portfolio built to showcase my work in embedded systems, hardware-focused engineering, low-level programming, and software development.

## Live Site
[View Portfolio](https://kaiden-bell.github.io/Portfolio/)

## About
This portfolio serves as a central place to highlight my technical projects, engineering background, and development experience. It was designed to reflect both the visual side of frontend development and the systems-oriented mindset behind my work.

## Built With
- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Features
- Responsive modern UI
- Animated transitions and interactive sections
- Project showcase for engineering and software work
- Skills section for hardware and software capabilities
- GitHub Pages deployment
